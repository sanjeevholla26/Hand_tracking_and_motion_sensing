User Guide

Step 1

Register and login to the website

Step 2

Map the hand gestures with the mouse actions of your chooise, See the mappings and also you can delete them if you wish. Go through the details of each Hand gesture and mouse actions in their respective pages, to get to know about them in detail. You can update the mappings and if the option which you want to map is blocked then delete the mapping which is having that option.

Step 3

Download the app - Once u click on the download, a zip file will be downloaded. Extract the contents of the zip file and then you can find the app in /mouse_action/dist/final_app with the name final_name. So double click to run the app

Step 4

In app there are two options/ways of using. There are default mappings which you can use. If you want to use your mappings which you created in website, then there is login option in the app. Enter the same credentials to login which you use for website. Then you can use your mappings

Step 5

If you want to stop using the app, then you can perform the action of showing the full palm to the camera and then closing it.
